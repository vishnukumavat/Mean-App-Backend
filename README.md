# Mean-App-Backend

start the mongodb server on your machine and create `analytics` database
in analytics create this collections [unique_global_visitors, website_visits]

Open terminal in the cloned directory

`cd Mean-App-Backend`
`npm install`

dir structure :

Mean-App-Backend
|----model
| |-> connection.js
| |-> index.js
| |-> models.js
|----node_modules
| |---(dependency_files)
|----.gitignore
|----index.js
|----package-lock.json
|----package.json
|----README.md
|----service_worker.js
|----service.js

`npm start`
